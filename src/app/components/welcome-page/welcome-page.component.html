Welcome

Welcome to my Deck Builder app!

Try it out! You can browse cards and build a deck for heroes of your choice. All cards are composed dynamically from data using HTML / CSS.

Current issues: object positioning on iOS.

Project history: this was my first own programming project as student, originally written in Java / JavaFX in many iterations for the purpose of learning programming and relevant development aspects related to a large project. The prototypes contained a sort-of functional logic with an interactive UI featuring a variety of objects and highlighting for attacks, spells and abilities. However, the scope of the project proved to be too large and the project was not finished. This application revives the original prototype to the web platform and includes the deck building feature that was integral to the game.

Note: the application is currently under active development as changes can occur on a daily basis.
